CSS Animations Showcase

Day 7 Tasks - Master CSS Animations & Transitions

1

Change Color on Hover

A button that changes color smoothly when hovered over using CSS transitions.

.hover-button { transition: all 0.3s ease; background: #667eea; } .hover-button:hover { background: #f093fb; transform: translateY(-3px); }
2

Animate a Moving Box

A div that moves from left to right across the screen using CSS animations.

@keyframes moveLeftRight { 0%, 100% { transform: translateX(-100px); } 50% { transform: translateX(100px); } } .moving-box { animation: moveLeftRight 3s ease-in-out infinite; }
3

Pulse Effect

An icon with a pulsing effect that grows and shrinks using CSS animations.

@keyframes pulse { 0%, 100% { transform: scale(1); opacity: 1; } 50% { transform: scale(1.2); opacity: 0.8; } } .pulse-icon { animation: pulse 2s ease-in-out infinite; }
4

Rotating Loader

A circular loader that rotates continuously using CSS keyframes.

@keyframes rotate { 0% { transform: rotate(0deg); } 100% { transform: rotate(360deg); } } .rotating-loader { animation: rotate 1s linear infinite; border: 4px solid rgba(102, 126, 234, 0.2); border-top: 4px solid #667eea; }
5

Fade In Text

Text that fades in from transparent to opaque when the page loads.

Welcome to CSS!
@keyframes fadeIn { 0% { opacity: 0; } 100% { opacity: 1; } } .fade-in-text { animation: fadeIn 3s ease-in; }
6

Bouncing Ball

A ball that bounces up and down using CSS keyframes animation.

@keyframes bounce { 0%, 100% { transform: translateY(0px); animation-timing-function: ease-out; } 50% { transform: translateY(-80px); animation-timing-function: ease-in; } } .bouncing-ball { animation: bounce 1.5s ease-in-out infinite; }
7

Transition on Width Change

A div that expands in width smoothly when hovered over.

Hover to Expand
.expanding-div { width: 150px; transition: width 0.5s ease; } .expanding-div:hover { width: 300px; }
8

Flip Card Animation

A card that flips to reveal content on the other side on hover.

Front Side
Back Side!
.flip-card-inner { transition: transform 0.8s; transform-style: preserve-3d; } .flip-card:hover .flip-card-inner { transform: rotateY(180deg); } .flip-card-back { transform: rotateY(180deg); }
9

Delayed Animation

Three boxes with animations that start with slight delays after each other.

.delayed-box { animation: slideUp 2s ease-out infinite; } .delayed-box:nth-child(1) { animation-delay: 0s; } .delayed-box:nth-child(2) { animation-delay: 0.3s; } .delayed-box:nth-child(3) { animation-delay: 0.6s; }
10

Infinite Scroll Animation

Background text that continuously scrolls horizontally, creating a marquee effect.

🚀 Infinite Scrolling Text Animation • CSS Mastery • Web Development • 🚀
@keyframes scroll { 0% { transform: translateX(100%); } 100% { transform: translateX(-100%); } } .scroll-text { animation: scroll 8s linear infinite; white-space: nowrap; }
